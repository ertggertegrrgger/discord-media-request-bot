# Discord Media Request Bot

Бот для обработки заявок на роль media через систему тикетов.

## Функции

- Отправка сообщения с кнопкой подачи заявки
- Создание приватных тикет-каналов
- Автоматическая нумерация тикетов
- Проверка на активные тикеты
- Форма заявки с 7 вопросами
- Система принятия/отклонения заявок
- Автоматическое удаление тикетов

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Настройте `.env` файл:
```
DISCORD_TOKEN=ваш_токен_бота
REQUEST_CHANNEL_ID=1495133736940077187
MEDIA_ROLE_ID=1495138025422852126
IMAGE_PATH=media-image.jpg
```

3. Добавьте изображение `media-image.jpg` в папку бота

4. Зарегистрируйте slash команду `/ticket` в Discord Developer Portal

5. Запустите бота:
```bash
npm start
```

## Использование

### Для пользователей:
1. Нажмите кнопку "ᴨодᴀᴛь зᴀяʙᴋу" в канале заявок
2. Заполните форму в созданном тикете
3. Дождитесь решения от media requester

### Для media requester:
1. Зайдите в тикет-канал
2. Используйте команду `/ticket`
3. Нажмите "Принять тикет" или "Отклонить"
4. Нажмите "Удалить тикет" для закрытия

## Требования

- Node.js >= 18.0.0
- Discord.js v14
